# XsltDebugger.DebugAdapter

This is the .NET Debug Adapter Protocol (DAP) server for the XSLT Debugger VS Code extension.

## Project Setup

This directory will contain the C# project implementing the DAP server for XSLT debugging.

- Language: C# (.NET 8 or latest LTS)
- Protocol: Debug Adapter Protocol (DAP)
- Features: XSLT execution, breakpoints, stepping, variable/context inspection
